# ZBoost: Advanced libraries for Zig language.

## Concept
This project is a set of libraries for the Zig language that offers a lot of resources to extend and simplify the practical use of the Zig standard libraries. Its purpose is very similar to the Boost libraries for C++, but for Zig.

## Resource Scopes
- Networking
    - HTTP
    - TCP/UDP
    - Sockets
    - Routing and Middleware
    - Connection Security
- Concurrency
    - Futures
    - Semaphores
    - Thread Pooling
    - Multiprocessing
    - Message Passing
    - Asynchronous Event Handling
- Math
    - Linear Algebra
    - Statistics and Probability
    - Advanced Calculus
- AI
    - Machine Learning
    - Tensors
    - Reinforcement Learning
- Collections
    - Stacks
    - Queues
    - Trees and Graphs
- System
    - OS Functions
    - Args and Envs
- Misc
    - Strings
    - Traits and Helper Types
    - Metaprogramming
    - Memory Interfaces
